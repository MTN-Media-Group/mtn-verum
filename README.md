# mtn-verum

Invisible, keyed, machine-readable image marking for Go.

`mtn-verum` writes a cryptographically-keyed mark directly into image pixels so images generated by an MTN service can be identified later, without relying on metadata that most uploaders strip.

```
go get github.com/MTN-Media-Group/mtn-verum
```

## Quick start

```go
import (
    "context"
    "os"
    "time"

    verum "github.com/MTN-Media-Group/mtn-verum"
)

key := verum.Key{ID: "k1", Secret: []byte("a high-entropy secret of at least 32 bytes")}
cfg := verum.Config{
    ActiveKey:     key,
    DetectionKeys: []verum.Key{key},
    Strength:      verum.StrengthBalanced,
}

payload := verum.Payload{
    GeneratedAt:  time.Now(),
    Provider:     "mtnai",
    Model:        "flux.1-pro",
    GenerationID: "gen_abc123",
}

src, _ := os.ReadFile("input.png")
res, err := verum.Embed(context.Background(), src, "image/png", payload, cfg)
// res.Data carries the marked image; res.Quality has SSIM, PSNR, MaxDelta.

det, err := verum.Detect(context.Background(), res.Data, "image/png", cfg)
// det.Detected, det.KeyID, det.PayloadDigest, det.Confidence.
```

`Verify` is `Detect` plus a digest check against an expected `Payload`.

## Capabilities

| Capability                                   | v1 |
| -------------------------------------------- | -- |
| PNG embed/detect at native scale             | ✓  |
| JPEG embed/detect at quality 95 (Robust)     | ✓  |
| WebP decode and lossless encode (no CGO)     | ✓  |
| Multi-key detection / key rotation           | ✓  |
| Premultiplied alpha round-trip preserved     | ✓  |
| Fully transparent pixels untouched           | ✓  |

## Not yet supported

- JPEG quality below 95 is not calibrated.
- Cropping breaks detection (the tile grid is not crop-resilient).
- Resize and downscale break detection (no scale-sync header).
- Reed–Solomon ECC. v1 uses bit majority + CRC32.
- Lossy WebP output (the project is pure-Go; no CGO bindings to libwebp).

## Strength profiles

| Profile      | Default delta | Default min SSIM | Default min PSNR |
| ------------ | ------------- | ---------------- | ---------------- |
| Invisible    | 4             | 0.999            | 50 dB            |
| Balanced     | 6             | 0.997            | 46 dB            |
| Robust       | 9             | 0.993            | 42 dB            |

`Embed` retries with successively weaker profiles if the requested gates fail.
