# mtn-verum

Invisible, keyed, machine-readable image identification for Go.

`mtn-verum` embeds and detects a small cryptographically-keyed mark in image
pixels so that images generated by an MTN service can later be identified as
such, without relying on metadata that strips during re-encoding or upload.

Status: pre-release. API and on-image format are not yet stable.
